# 📊 AdventureWorks Sales Variance Analysis

## 🎯 Project Overview
This project focuses on analyzing sales performance against budget using the AdventureWorks dataset. I developed an interactive Power BI dashboard to identify key variances and top-performing segments.

## 🚀 Key Features
* **Variance Tracking:** Calculated Year-to-Date (YTD) Sales vs. Budget using custom DAX measures.
* **Top 10 Analysis:** Dynamically filtered views for Top 10 Products and Top 10 Customers by total revenue.
* **Visual Design:** Created a clean, branded UI with a custom layout for high-level KPI tracking.
